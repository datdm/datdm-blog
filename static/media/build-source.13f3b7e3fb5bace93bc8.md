# Config source React Native

<a name="readme-top"></a>
Tutorial config React Native.

## 1. Expo CLI

### Installation

- Install global expo cli
  ```sh
  npm install -g expo-cli
  ```
- Initial project
  ```sh
  expo init react-native
  ```
- Run
  ```sh
  yarn start
  yarn run androind
  yarn run web
  ```
